知识点：

1.angular只能有一个入口，用ng-app来进行设置

2.angular的表达式的格式为{ { } }

3.angular表达式可以进行计算或取值、定义变量和赋值

4.angular里ng-什么什么，都是属于angular里自带的指令-directive

5.ng-init指令定义变量,最好不要在表达式里定义，因为表达式的作用一般来说是显示数据

6.可以通过ng-bind指令来给标签绑定数据，这样的话就算angularjs放在页面底部加载也不会出现双花括号的闪烁，会把标签的原有内容替换

效果：

计算：{{1+2}} - - {{2*4}} - {{4/2}} -- {{10%3+6}}

取值：{{[1,2,3,4,5][3]}}--{{{name:'allen'}.name}}

定义数据：{{age=18}} -- {{age}} --{{'male'}} -- {{sex}} --{{hobby}}
